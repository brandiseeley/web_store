Web Store API

RESTful API for managing products in an online store

Getting Started

Base URL: /v1
Format: JSON
Authentication: Token-based (required for write operations)

Authentication

For creating, updating, or deleting resources, include the following header:

Authorization: token AUTH_TOKEN

Endpoints

GET /v1/products

Fetch all products in the store.

Example Request:

curl https://your-domain.com/v1/products

Example Response:

[ { "id": 1, "name": "Magic Pen", "sku": "magp100", "price": 500 }, ... ]
GET /v1/products/:id

Fetch a single product by ID.

Path Parameters:

id integer required

Product ID

Example Request:

curl https://your-domain.com/v1/products/1

Example Response:

{ "id": 1, "name": "Magic Pen", "sku": "magp100", "price": 500 }

Error Responses

Status Code Description
401 Unauthorized - Authentication required or invalid credentials
404 Not Found - Resource doesn't exist
415 Unsupported Media Type - Content-Type must be application/json
422 Unprocessable Entity - Validation error

Error Response Format:

{ "status_code": 422, "message": "Name can't be blank" }
